Operasi Pada String Itu Apa?





Dalam belajar bahasa python terdapat fungsi bawaan yang harus kamu ketahui.

fungsi ini yang digunakan untuk melakukan beberapa operasi yang melibatkan string.

tahu tidak apa itu string?

String adalah bentuk data yang biasa dipakai dalam bahasa pemrograman untuk keperluan menampung dan memanipulasi data teks, misalnya untuk menampung (menyimpan) suatu kalimat.

Nah, fungsi bawaan yang dimaksud ada di bawah ini :
1. Fungsi Upper() dan Lower()
Kedua fungsi ini digunakan untuk mengubah ukuran huruf menjadi huruf kapital dan huruf kecil. Untuk mengubah ke huruf kapital digunakan fungsi upper(), sedangkan jika ingin mengubah ke huruf kecil gunakan fungsi lower().

2. Fungsi Len()

Fungsi len() digunakan untuk mengetahui panjang dari suatu string.

3. Fungsi mengatur rerata teks


Untuk merapikan output dari kalimat kita bisa menggunakan beberapa fungsi bawaan dari string. Untuk mengatur rata tengah kamu dapat menggunakan center(). Untuk mengatur rata kanan gunakan rjust(). Untuk mengatur rata kiri gunakan ljust().

4. Fungsi Join() dan Split()

Fungsi join() digunakan untuk menggabungkan string yang terdapat dalam list, tuple ataupun set. Kebalikan dari fungsi join() adalah split(), digunakan untuk memisahkan string.

5. Fungsi Index()

Fungsi index() digunakan untuk mengetahui posisi indeks pertama suatu karakter pada string

6. Fungsi Replace()

Fungsi replace() digunakan jika ada karakter atau bagian dari suatu string yang ingin diganti.

Artikel ini berdasarkan modul Tipe Data dan Operator. Pelajari selengkapnya di bawah ini.
