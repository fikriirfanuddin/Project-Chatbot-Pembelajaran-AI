Tipe Data dan Operator Itu Apa?





Pemrograman Python mempunyai beberapa komponen penting.

Apa saja komponen penting yang dimaksud?

Jawabannya adalah tipe data dan operator.

Apa Itu Tipe Data Python?

Tipe data python merupakan komponen yang berfugsi untuk memberikan keterangan data yang diproses dalam bahasa python.

Bahasa Python memiliki beberapa macam tipe data:
1. Integer dan Float
integer(int) merupakan nilai dari bilangan bulat, dan float merupakan nilai bilangan desimal.

2. String

String merupakan tipe data jika kamu ingin menggunakan nilai berupa huruf, angka atau kalimat. Dalam python String kamu bisa menulis dengan kode str. Untuk menggunakan tipe data ini harus diapit oleh tanda petik (‘ atau “) di awal dan di akhir kalimat

3. Boolean


Boolean merupakan tipe data yang mempunyai nilai True dan False. Dimana True mempunyai nilai 1, dan False mempunyai nilai 0.

Apa Itu Operator ?

Operator disini disebut sebagai operator aritmatika. Operator yang digunakan dalam bahasa python melingkupi tambah, kurang, kali, dan bagi. Pada python operator aritmatika biasanya dilambangkan dalam 1 atau 2 karakter.
